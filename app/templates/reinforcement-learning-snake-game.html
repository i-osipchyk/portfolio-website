Introduction

When I started learning LLMs, I first faced the concept of Reinforcement Learning in practice. That is why I've decided to familiarize myself with how it works in practice. The best way to do it is to implement some simple agent by myself. That is why I've decided to do this project. Later I want to continue my journey in RL with multi-agent environment, for example tic-tac-toe game.

Description

During the game, the agent discovers correct moves through the reward. If the snake eats an apple, it gets 10. If it touches itself or the wall, it gets -10. In all other cases it gets 0. Snake knows if there is a wall on the right, on the left or right after its head. It also knows about the position of the apple relative to its head.
